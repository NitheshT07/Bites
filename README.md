# 🍔 MK Little Bites — Premium Dining & Celebrations

> **"The Taste That Brings People Together."**  
> A high-end, responsive landing page for Thokkavadi's favorite dining spot.

---

## ✨ Features

- 🎬 **Cinematic Hero**: High-fidelity animations with Framer Motion.
- 🍽️ **Interactive Menu**: Full categorized menu with a smooth modal experience.
- 🍦 **Sweet Spot**: Dedicated sections for **Premium Ice Creams** & **Pastries**.
- 🌟 **V3.0 Masterpiece UI**: Professional luxury aesthetics with glassmorphism and gold accents.
- 📱 **Fully Responsive**: Optimized for every device, from mobile to desktop.
- 💬 **Instant Connectivity**: Direct WhatsApp integration for quick orders and bookings.

## 🚀 Getting Started

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/NitheshT07/Bites.git
cd Bites
```

### 2️⃣ Install Dependencies
```bash
npm install
```

### 3️⃣ Run Locally
```bash
npm run dev
```
